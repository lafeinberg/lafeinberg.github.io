Game Design Projects
Spy Quest: Platformer

By: Aditi Dam, Luci Feinberg, Luiza Leschziner, Shivani Patel

With a group of 4 for Digital Game Design course COMS BC3997, we designed and created a captivating 3-level immersive 2D platform game using Unity, implementing advanced AI algorithms, rangefinder sensors, and procedural content generation (PCG) techniques for environment generation and autonomous agent behavior in C#. Additionally, I prioritized inclusivity and accessibility by incorporating features specifically designed for color-blind and hearing-impaired players, such as color-grading options and screen-shake effects, ensuring that everyone can enjoy the game to its fullest potential.

Read more
Barnard Mapping Game

By: Luci Feinberg

A prototype immersive game has been developed for incoming Barnard students to engage with Columbia's campus map. This innovative game employs rangefinder sensors to create autonomous agents that simulate students navigating and locating their classes. It also includes speech, to have users interact with agents with "trivia" questions. After undergoing playtesting with five first-year students, the product is now in the process of being expanded for broader use. Coded using C++ in Unity 2D, looking to traslate code to 3D Unity.

Covid Spreader: Sprite Pachinko

By: Luci Feinberg

Through Unity's extensive learning program, I successfully prototyped a 2D Unity game. I completed the Unity Essentials Pathway, covering real-time 3D, Unity programming, and 2D game development, including the Sprite Pachinko challenge. Utilizing the skills and knowledge gained, I developed a 2D Unity game focused on COVID-19's spread, challenging players to manage its propagation.
